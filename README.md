# 🧠 NeuroDesk

**NeuroDesk** is an AI-powered churn risk analyzer and decision-feed dashboard tailored for gym franchises. Built with a modern Next.js frontend and a FastAPI backend, NeuroDesk leverages **NVIDIA NIM (Llama 3.3 70B)** to generate actionable insights and personalized member retention strategies in real-time.

## 🚀 Features
- **AI Decision Feed:** Generates dynamic, context-aware retention strategies (e.g. at-risk corporate batches, upselling opportunities).
- **Action Automation:** "One-click" approval to dispatch personalized WhatsApp messages to at-risk members.
- **Location Status:** Real-time health metrics (members, revenue) across multiple branches.
- **NVIDIA NIM Integration:** Uses `meta/llama-3.3-70b-instruct` to process complex historical churn data and output structured JSON directives.
- **Intelligent Caching:** Fast, cached API responses with manual bypass (`?refresh=true`) for live inference demos.

## 🏗️ Architecture
- **Frontend:** Next.js (React), TailwindCSS, Lucide Icons, Glassmorphism UI.
- **Backend:** Python FastAPI, Uvicorn, OpenAI Python SDK (configured for NVIDIA NIM).
- **AI Layer:** NVIDIA NIM APIs (`meta/llama-3.3-70b-instruct`) for rapid, structured inference.
- **Data & Caching:** JSON-based state management and file-based API caching for resilience.

## 🛠️ How it Works & How to Use It

### 1. Backend Setup
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install fastapi uvicorn openai python-dotenv pydantic
```
Create a `.env` file in the `backend/` directory:
```
NVIDIA_API_KEY=your_nvidia_nim_api_key_here
```
Run the backend:
```bash
uvicorn main:app --reload --port 8000
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Open `http://localhost:3000` in your browser.

### 3. Usage
- The dashboard automatically loads location metrics and the AI decision feed.
- The **Critical** card is dynamically generated by the NVIDIA NIM API based on current gym data.
- Click **Approve Action** on any card to simulate dispatching the AI-generated personalized WhatsApp message. Expand **Explain More** to view the LLM's chain of thought.
